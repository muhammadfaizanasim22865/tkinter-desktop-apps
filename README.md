## 🔍 App Details

### 🔢 Calculator
A clean desktop calculator built with Tkinter. Implements **operator overloading** using Python OOP — goes beyond a basic calculator to demonstrate how Python's dunder methods (`__add__`, `__sub__`, etc.) can be applied in a real GUI application.

---

### 📊 DSA Tracker
A personal productivity tool to track progress through Data Structures & Algorithms topics. Built out of a real need — I used this during my own DSA learning journey. Demonstrates practical application of Tkinter for building tools that actually solve your own problems.

---

### 🍅 FizzPomo
A personalized **Pomodoro timer** desktop app — named after my own nickname "Fizzy." Implements the Pomodoro technique (25 min work / 5 min break cycles) to boost focus and productivity. A tool I built and actually use while studying.

---

### 🔐 FizzyCrypt
A desktop application for **encrypting and decrypting messages** securely. Allows users to encode sensitive text into a cipher and decode it back. Built with a focus on privacy and security concepts — an early exploration into cryptography through practical application.

---

### 🔑 Secret Calculator
The most creative app in this collection — a calculator on the surface, but enter a secret code and it **reveals a hidden password vault** underneath. Inspired by real-world security through obscurity concepts. Demonstrates how UI/UX and logic can be layered to create unexpected functionality.

## 🚀 How to Run Any App

**Prerequisites:** Python 3 installed on your machine. Tkinter comes built-in with Python — no extra installation needed.

```bash
# Clone the repo
git clone https://github.com/muhammadfaizanasim22865/tkinter-desktop-apps

# Navigate into any app folder
cd tkinter-desktop-apps/fizzycrypt

# Run the app
python fizzycrypt.py
```
